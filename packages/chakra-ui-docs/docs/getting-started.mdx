# Getting Started

Install Chakra UI and it's peer dependencies
<br />

<Code isReadOnly lang="bash" code={`yarn add @chakra-ui/vue emotion`}
/>

or (with npm)
<br />
<Code isReadOnly lang="bash" code={`npm install @chakra-ui/vue emotion --save`}
/>

### With Nuxt.js
> **Note:**<br />
> If you're using Nuxt, you need to install `@nuxtjs/emotion` package as well to server-side render your styles.

<br />

<Code isReadOnly lang="bash" code={`yarn add @chakra-ui/vue emotion @nuxtjs/emotion`}
/>

## Usage
**1. Import the Chakra UI plugin in your `main.js` file.**
<br />
<Code isReadOnly lang="js" code={`import Vue from 'vue'
import Chakra from '@chakra-ui/vue'
import App from './App.vue'\n
Vue.use(Chakra)\n
new Vue({
  el: '#app',
  render: (h) => h(App)
}).$mount()`} />

<!-- For some reason something breaks ehere -->
**2. Wrap your application inside the Chakra `ThemeProvider`.** 
We also recommend that you include the `CSSReset` component to normalize all browser styling.
_In your `App.vue` file._
<br />
<Code isReadOnly lang="vue" code={`<template>
  <ThemeProvider>
    <CSSReset />
    <!--
      Your application code goes here! üòÅ
    -->
  </ThemeProvider>
</template>

<script>
  import { ThemeProvider, CSSReset } from '@chakra-ui/vue'

  export default {
    name: 'App',
    components: {
      ThemeProvider,
      CSSReset
    }
  }
<\/script>
`} />
