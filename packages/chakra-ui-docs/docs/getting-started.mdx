# Getting Started

Install Chakra UI and it's peer dependency, `emotion`
<br />

<Code isReadOnly lang="bash" code={`yarn add @chakra-ui/vue emotion`}
/>

or (with npm)
<br />
<Code isReadOnly lang="bash" code={`npm install @chakra-ui/vue emotion --save`}
/>

### With Nuxt.js
> **Note:**<br />
> If you're using Nuxt, you need to install `@nuxtjs/emotion` package as well to server-side render your styles.

<br />

<Code isReadOnly lang="bash" code={`yarn add @chakra-ui/vue emotion @nuxtjs/emotion`}

## Usage
**1. Import the Chakra UI plugin in your `main.js` file.**

In order to use Chakra, you need to wrap your main application inside the Chakra `ThemeProvider` component.
You can do so as shown below.

<br />
<Code isReadOnly lang="js" code={`import Vue from 'vue'
import Chakra, { ThemeProvider } from '@chakra-ui/vue'
import App from './App.vue'\n
Vue.use(Chakra)\n
new Vue({
  el: '#app',
  render: (h) => h(ThemeProvider, [
    h(App)
  ])
}).$mount()`} />


If you'd like to add extend the Chakra theme with custom brand colors you can do so as shown below. This is also optional.

<br />
<Code isReadOnly lang="js" code={`import Chakra, { ThemeProvider, theme } from '@chakra-ui/vue'
import customTheme from 'my-custom-theme.js'\n
Vue.use(Chakra, {
  theme: {
    ...theme,
    colors: {
      ...theme.colors,
      brand: {
        900: "#1a365d",
        800: "#153e75",
        700: "#2a69ac",
      },
    }
  }
})\n
new Vue({
  el: '#app',
  render: (h) => h(ThemeProvider, [
    h(App)
  ])
}).$mount()`} />


## Injecting Global Styles.
Sometimes you may need to apply css reset styles to your application.
Chakra UI exports a `CSSReset` that'll remove browser default styles. It's heavily inspired by [Tailwind's preflight](https://unpkg.com/tailwindcss@1.1.2/dist/base.css)

> üö® We highly recommend that you add the `CSSReset` at the root of your application to ensure all components work correctly.

<br />
<Code isReadOnly lang="js" code={`import Vue from 'vue'
import Chakra, { ThemeProvider, CSSReset } from '@chakra-ui/vue'
import App from './App.vue'\n
Vue.use(Chakra)\n
new Vue({
  el: '#app',
  render: (h) => h(ThemeProvider, [
    h(CSSReset),
    h(App)
  ])
}).$mount()`} />
